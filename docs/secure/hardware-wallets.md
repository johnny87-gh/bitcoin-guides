# Hardware Wallets

Hardware wallets are physical devices that store your Bitcoin private keys offline, providing the highest level of security for your digital assets.

## What are Hardware Wallets?

Hardware wallets are dedicated, tamper-resistant devices that:
- Store private keys offline (cold storage)
- Sign transactions internally without exposing keys
- Provide physical verification of transactions
- Protect against malware and hacking
- Enable recovery via backup seed phrase

## Why Use a Hardware Wallet?

### Maximum Security
- Private keys never leave the device
- Immune to computer viruses and malware
- Physical device required for transactions
- PIN protection and tamper detection
- Recovery options available

### Peace of Mind
- Your Bitcoin is under your control
- No dependence on third parties
- Protection from online threats
- Insurance against exchange failures
- Long-term storage solution

## Hardware Wallet Features

### Security Features

#### Secure Element
- Tamper-resistant hardware
- Physical protection against extraction
- Common in Ledger and some other wallets
- Military-grade security standards

#### PIN Protection
- Required to access wallet
- Multiple failed attempts trigger reset
- Device wipes after repeated failures
- Choose strong PINs

#### Passphrase Option
- Adds additional layer of security
- Creates hidden wallets
- Remember passphrase to access funds
- Optional advanced feature

#### Recovery Phrase
- 12-24 words for backup
- Written on physical medium
- Critical for wallet recovery
- Never store digitally

### Connectivity Options

#### USB Connection
- Most common method
- Desktop software required
- Reliable and fast
- Some models support USB-C

#### Bluetooth
- Mobile-friendly
- Ledger Nano X feature
- Encrypted communication
- Convenience vs security tradeoff

#### Air-Gapped (QR Codes)
- Maximum security
- No physical connection needed
- Keystone, Coldcard support
- More complex workflow

#### MicroSD Card
- Coldcard feature
- Encrypted backup storage
- Offline storage of transactions
- Advanced use case

## Setting Up Your Hardware Wallet

### Initial Setup Process

#### Step 1: Unboxing and Verification
- Verify package integrity
- Check security seals
- Ensure device hasn't been tampered with
- Download software from official website only

#### Step 2: Initialization
- Connect device to computer
- Follow on-screen instructions
- Set up PIN code
- Write down recovery phrase

#### Step 3: Recovery Phrase Backup
- Write down 12-24 words on provided card
- Never store digitally or take photos
- Make multiple copies
- Store securely in different locations

#### Step 4: Software Installation
- Install official wallet software
- Verify software authenticity
- Set up software wallet
- Connect hardware wallet

#### Step 5: Test Transaction
- Send small test transaction
- Verify everything works
- Check fee calculation
- Confirm receiving address

## Best Practices

### Security Practices
- Never share recovery phrase or PIN
- Keep device firmware updated
- Use strong PIN (not 0000, 1234, etc.)
- Verify all transaction details on device
- Consider passphrase for additional security
- Use antivirus software on host computer

### Storage Practices
- Store device securely when not in use
- Keep recovery phrase in multiple secure locations
- Consider using fireproof and waterproof storage
- Never store recovery phrase digitally
- Share recovery phrase only with trusted individuals

### Usage Practices
- Verify transaction details on device display
- Use different addresses for each transaction
- Regularly backup recovery phrase
- Test recovery process with small amounts
- Monitor transaction history

### Backup Practices
- Multiple copies of recovery phrase
- Store in different physical locations
- Consider metal backup options
- Check recovery phrase periodically
- Update backups if wallet changes

## Common Mistakes to Avoid

- Buying used or tampered devices
- Storing recovery phrase digitally
- Using weak PIN codes
- Not testing recovery phrase
- Sharing device or recovery phrase
- Using unofficial firmware or software
- Not keeping firmware updated
- Not verifying transaction details
- Losing recovery phrase without backup
- Exposing recovery phrase to photos or cameras

## Troubleshooting

### Device Issues
- Restart device if unresponsive
- Update firmware if available
- Try different USB cable or port
- Contact manufacturer support

### Software Issues
- Reinstall wallet software
- Check for software updates
- Try different computer
- Clear cache and cookies

### Recovery Issues
- Double-check recovery phrase spelling
- Ensure words are from correct wordlist
- Try different word order (if you changed it)
- Contact support if multiple attempts fail

### Transaction Issues
- Verify sufficient funds
- Check network fees
- Confirm transaction hasn't been double-spent
- Check mempool congestion
