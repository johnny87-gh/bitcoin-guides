# Software Wallets

Software wallets are applications that run on your computer or smartphone, offering convenience and flexibility for everyday Bitcoin use.

## What are Software Wallets?

Software wallets are applications that:
- Run on your devices (computer, phone, tablet)
- Store private keys encrypted on your device
- Provide convenient access to Bitcoin
- Offer various features and integrations
- Balance security and usability

## Types of Software Wallets

### Desktop Wallets
Run on your computer (Windows, Mac, Linux)
- More storage and processing power
- Often more features than mobile wallets
- Can connect to hardware wallets
- Better for larger amounts
- Require regular backups

### Mobile Wallets
Run on smartphones (iOS, Android)
- Maximum convenience and portability
- Always with you for transactions
- Touch ID/Face ID integration
- Great for daily spending
- Smaller screen limits complexity

### Web Wallets
Run in your web browser
- Most convenient, least secure
- Access from any device
- Browser-based encryption
- Use only for small amounts
- Higher security risk

## Top Desktop Wallets

### Electrum
**Best Overall Desktop Wallet**

- Lightweight and fast
- Supports hardware wallets
- Advanced features (coin control, RBF)
- Highly customizable
- Long-standing reputation
- Open source

**Best for**: Desktop users wanting features and control

**Security**: Encrypted wallet file, optional 2FA, multisig support

### Sparrow Wallet
**Advanced User's Choice**

- Based on Electrum codebase
- Extensive privacy features
- CoinJoin support
- Coldcard integration
- Rich functionality
- Open source

**Best for**: Advanced users and privacy advocates

**Security**: Strong encryption, privacy-focused, multisig support

### Wasabi Wallet
**Privacy-Focused Wallet**

- Built-in CoinJoin mixing
- Automatic coin control
- Strong privacy features
- Simple interface
- Open source
- Tor integration

**Best for**: Users prioritizing privacy

**Security**: CoinJoin, Tor, coin control, encrypted storage

### Bitcoin Core (Full Node)
**Most Secure Software Wallet**

- Full validation of all transactions
- Maximum privacy and security
- No reliance on third parties
- Entire blockchain downloaded (400GB+)
- Not for beginners
- Resource-intensive

**Best for**: Technical users wanting maximum security

**Security**: Full validation, no SPV, complete control

## Security Best Practices

### General Security
- Enable biometric authentication when available
- Use strong PIN codes
- Keep wallet software updated
- Backup recovery phrases properly
- Never share private keys

### Desktop Wallet Security
- Use antivirus software
- Keep operating system updated
- Use secure network connections
- Consider using with hardware wallet
- Regular backups

### Mobile Wallet Security
- Enable device lock screen
- Use biometric authentication (fingerprint, Face ID)
- Keep OS updated
- Avoid public Wi-Fi for transactions
- Use VPN when on public networks

### Recovery Security
- Write down recovery phrase (seed words) on paper
- Never store recovery phrase digitally
- Make multiple copies of recovery phrase
- Store securely in different locations
- Test recovery with small amounts

## Common Software Wallet Mistakes

- Not backing up recovery phrase
- Storing recovery phrase digitally
- Using weak PIN or passwords
- Not testing recovery procedure
- Leaving large amounts on mobile wallets
- Sharing recovery phrase or screenshots
- Using unverified wallet software
- Not keeping wallet updated
- Not verifying transaction details
- Ignoring security warnings

## Backup and Recovery

### Creating a Backup
1. **Initialize wallet**: Set up new wallet
2. **Write down seed phrase**: Carefully write all words
3. **Verify words**: Double-check spelling and order
4. **Multiple copies**: Create several physical copies
5. **Secure storage**: Store in safe, separate locations
6. **Test recovery**: Practice recovering with small amount

### Recovery Process
1. **Install wallet**: On new device
2. **Choose restore**: Select "Restore from seed" option
3. **Enter seed words**: In correct order
4. **Verify words**: Confirm word spelling
5. **Wait for sync**: Allow wallet to sync
6. **Verify funds**: Check balance matches

## Integration with Hardware Wallets

Many software wallets integrate with hardware wallets:
- **Electrum**: Supports Ledger, Trezor, Coldcard, and more
- **Sparrow**: Excellent Coldcard integration
- **BlueWallet**: Supports Ledger and Trezor
- **Wasabi**: Supports some hardware wallets

### Benefits of Integration
- Software convenience
- Hardware security
- Best of both worlds
- Use hardware for storage
- Use software for transactions

## Advanced Features

### Coin Control
- Manually select which coins to spend
- Improve privacy by using newer coins
- Control transaction fees
- Consolidate UTXOs

### Replace-by-Fee (RBF)
- Speed up stuck transactions
- Increase fee after sending
- Only if recipient supports
- Available in advanced wallets

### Child Pays For Parent (CPFP)
- Alternative to RBF
- Spend from child transaction
- Increase fee to speed up parent
- Requires advanced wallet

### Multi-Signature
- Require multiple signatures to spend
- Enhanced security
- More complex setup
- Requires multiple devices or wallets

## Lightning Network in Software Wallets

### What is Lightning?
- Second layer for Bitcoin
- Instant, low-fee transactions
- Requires opening channels
- More complex than on-chain

### Lightning Wallets
- Phoenix: Lightning-only, simple
- Muun: Integrated Lightning
- BlueWallet: Supports Lightning
- Breeze: Lightning-focused

### Getting Started with Lightning
1. Choose Lightning wallet
2. Fund Lightning channel
3. Use for small transactions
4. Close channel when done
5. Watch channel fees

## When to Use Software Wallets

### Good For
- Everyday spending
- Small to medium amounts
- Convenience and accessibility
- Testing and learning
- Lightning Network transactions
- Mobile payments

### Use With Caution For
- Large amounts (use hardware wallet)
- Long-term storage
- Major life savings
- Business holdings

### Avoid For
- Very large holdings
- Entire Bitcoin savings
- Retirement funds (use cold storage)

## Summary

Software wallets offer convenience and functionality for everyday Bitcoin use. They're ideal for spending and small-to-medium amounts. For larger holdings or long-term storage, use a hardware wallet or combine software with hardware wallet integration.

**Best Practice**: Use a hardware wallet for savings (HODLing) and a software wallet for spending. Keep significant amounts in cold storage.
